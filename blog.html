Var : The var is the oldest keyword to declare a variable in JavaScript.By using Var we can update it's key value.

Let : The let is the updated keyword to declare a variable in JavaScript.let can be updated but not re-declared..

Const : Variables declared with the const maintain constant values.Const neither can be updated nor re-declareable.

Arrow functions a new feature introduced in ES6 in JavaScript. While both regular and arrow functions work in a similar manner, yet there are certain interesting differences between them. Regular function : Regular functions created using function declarations or expressions are constructible and callable. However, the arrow functions are only callable and not constructible.

Map :map makes an array unique generate a new array based on existing array.map modifies the array elements and save them into the new array.

forEach : Foreach takes a callback function and run that callback function on each element of array one by one.

filter : filter executes the callback and check its return value. filter does not update the existing array it will return a new filtered array every time.

find : find always pass a single element from an array .It starts a loop from the beginning. But it stops when the condition is matched.
